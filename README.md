# 🎬 Movie Data Analysis (QlikView)

## 📊 Overview
This project presents a data analysis of a movie dataset using QlikView.  
The goal was to explore relationships between budget, revenue, ratings, and other key factors.

## 🔍 Key Insights
- Comparison of Budget vs Box Office revenue
- Analysis of movie performance by genre
- Distribution of movies across different countries
- Overview of ratings and their impact

## 🛠 Tools Used
- QlikView
